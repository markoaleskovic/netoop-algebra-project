Algebra project for .NET OOP course

#Academic Context

This project was developed as a coursework assignment for Object-oriented programming in .NET environment, demonstrating proficiency in:

  - REST API consumption and JSON parsing with configurable data source modes (live API vs. offline files)
  - Cross-platform desktop development with WinForms and WPF sharing common data access layer
  - Asynchronous programming with responsive UI design across multiple resolutions
  - MVVM architecture and custom user controls with drag-and-drop functionality
  - Localization/globalization (Croatian and English language support)
  - File I/O for persistent configuration management
  - Design patterns and layered architecture principles
